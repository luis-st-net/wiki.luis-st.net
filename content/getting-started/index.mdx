## Prerequisites

- Node.js 18 or later
- npm or pnpm
- Git access to your documentation repository

## Installation

```bash
npm install
npm run dev
```

Once the development server starts, open [http://localhost:3000/wiki](http://localhost:3000/wiki) to browse the documentation experience.

## Directory Layout

```
content/
  index.mdx
  getting-started/
    index.mdx
    installation.mdx
    structure.mdx
```

Each folder becomes a collapsible section in the sidebar. Every `.mdx` file becomes a wiki page whose slug matches its file name.
